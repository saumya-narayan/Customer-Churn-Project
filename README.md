# Customer Churn Prediction Project

## 📌 Overview
This project analyzes customer churn data and predicts whether a customer will leave or not using Machine Learning.

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn

## 📊 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Model Training (Logistic Regression)
- Prediction & Accuracy Evaluation

## 📈 Results
- Built a churn prediction model
- Evaluated using accuracy and confusion matrix

## 🚀 Conclusion
Customers with low tenure and high monthly charges are more likely to churn.
